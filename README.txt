Online Resource 1 (ESM_1.zip)
=============================
Article : A Metadata-Only Feature-Augmented Method Factor for Ex-Post Correction
          and Attribution of Common Method Variance in Self-Report Health Instruments
Journal : Lifetime Data Analysis (submission)
Author  : Murat Yaslioglu, School of Business, Istanbul University, Istanbul, Turkiye
Contact : muratyas@istanbul.edu.tr

Contents
--------
famf.py                  Calibration library (Python 3; requires numpy, pandas, semopy)
sim_study.py             Monte Carlo simulation script (Section 6). Reproduces sim_results.csv.
                         Usage: python3 sim_study.py <start> <count> [reps] [outfile]
sim_results.csv          Replication-level results generated by sim_study.py
                         (7,200 model fits: 12 cells x 200 replications x 3 models)
empirical_bfi.py         Empirical analysis script (Section 7); uses the public SAPA 'bfi'
                         data via the pydataset package. Reproduces empirical_results.json.
empirical_results.json   Calibration and fit outputs of the empirical analysis
famf_sem.R               R/lavaan implementation of the full FAMF-SEM pipeline
FAMF_SEM_calculator.html Single-file browser calculator (no installation; data stay local)

The empirical dataset itself is publicly available as 'bfi' in the psych R package
(https://CRAN.R-project.org/package=psych) and via the pydataset Python package.
